# QuizApp
```bash
ng new QuizApp -sd client  --style scss --routing
```
# Generate components
```bash
ng g c register
ng g c navbar
ng g c quiz
ng g c result
ng g c shared
```
# Generate quize service
```bash
ng g s shared/quiz
```
